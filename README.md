## 🌟 特色功能

### 遊戲機制
- 5個獨特的關卡主題
- 每個關卡包含2張精美的卡牌
- 玩家可以從每個關卡中選擇並收集卡牌
- 關卡進度追蹤和完成度顯示

## 🎮 使用說明

1. 進入遊戲後，您會看到五個不同主題的關卡
2. 點擊任意關卡按鈕來選擇卡牌
3. 在彈出的對話框中選擇您喜歡的卡牌
4. 收集的卡牌會顯示在頂部的收藏區
5. 點擊收藏的卡牌可以全螢幕查看
6. 右上角有隱藏的重置功能（連續點擊三次）
7. 支援安裝為桌面應用（PWA）
